# whatsapp-fast-message
whatsapp-fast-message is a web tool that allows you to send messages to WhatsApp numbers without adding them to your contacts. Simply enter the phone number, write the message, and click the "Send Message" button. You will be redirected to WhatsApp Web, desktop, or mobile application available on your device.

## Features
- Quickly send messages to WhatsApp numbers without adding contacts.
- Automatic redirection to WhatsApp Web, desktop, or mobile.
- Simple and user-friendly interface.

## Requirements
- Modern web browser (Chrome, Firefox, Safari, etc.)
- Internet connection
